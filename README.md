#Jailbreak markup language

Jailbreak is a mark up language inspired by markdown.

## Guide

* Headings :

```$``` : Heading 1

``` $$ ``` : Heading 2

``` $$$ ``` : Heading 3

``` $$$$ ``` : Heading 4

``` $$$$$ ``` : Heading 5

``` $$$$$$ ```: Heading 6

* Listing :

```*``` : Bullet List

* Linking :

```[TITLE]-(URL)``` : Links

* Images :

``` img-URL-TITLE ``` : Image
